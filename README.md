\# 📊 Superstore Analytics Dashboard



\## 📌 Project Overview

This project is an interactive analytics dashboard built using HTML, CSS, and JavaScript. It analyzes the Superstore dataset to provide business insights through interactive visualizations, KPIs, and filters.



\---



\## 🚀 Features

\- Interactive dashboard

\- Year, Region, and Segment filters

\- Revenue and Profit analysis

\- Sales trend visualization

\- Category performance analysis

\- Customer segment analysis

\- Shipping mode analysis

\- Top states by revenue

\- Sub-category revenue and profit analysis



\---



\## 🛠️ Technologies Used

\- HTML5

\- CSS3

\- JavaScript

\- Superstore CSV Dataset



\---



\## 📊 Key KPIs

\- Total Revenue

\- Total Profit

\- Profit Margin

\- Total Orders

\- Units Sold
